# **Creando y eliminando BDs**

## Crear una base de datos

```sql
CREATE DATABASE [NOMBRE_BASE_DATOS];
```
Por ejemplo:
```sql
CREATE DATABASE mi_base_de_datos;
```

---

## Eliminar una base de datos

```sql
DROP DATABASE [NOMBRE_BASE_DATOS];
```
Por ejemplo:
```sql
DROP DATABASE mi_base_de_datos;
```
